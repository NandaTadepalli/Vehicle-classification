Model Info

Model: Vision Transformer (ViT) fine-tuned for vehicle classification.
Frameworks: Hugging Face Transformers, PyTorch, Flask.
Deployment: AWS EC2 for scalability and accessibility.
Features:

  • Upload vehicle images and get instant classification (car, bike, bus, van, etc.).
  • Deep learning models: CNN, ResNet, and ViT.
  • Future-ready: Easily extendable for number plate recognition, traffic monitoring, smart toll, and real-time edge AI.
  • Modern, responsive web interface with image preview and prediction results.
  • All uploaded images are securely stored and visible after upload and prediction.

How it works: The uploaded image is preprocessed and passed through the ViT model, which predicts the vehicle type. The result and the image are displayed instantly.
Tech Stack: Python, Flask, PyTorch, Hugging Face, AWS, HTML/CSS/JS.
